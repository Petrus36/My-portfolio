## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

- Update project information in `src/components/Projects.tsx`
- Modify personal information in `src/components/About.tsx`
- Update contact information in `src/components/Contact.tsx` and `src/components/Footer.tsx`
- Adjust colors and styling in `tailwind.config.js` and component files
